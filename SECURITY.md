# Security Policy

Release Doctor inspects package metadata and repository files. It should not execute target package scripts while diagnosing a project.

## Reporting a Vulnerability

Please report security issues privately through GitHub security advisories when available. If advisories are not enabled, contact the npm package maintainer.

Include:

- Affected version.
- Steps to reproduce.
- Impact.
- Suggested fix, if known.

Do not open public issues for vulnerabilities until a fix is available.
