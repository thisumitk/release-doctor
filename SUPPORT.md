# Support

Use these channels for Release Doctor support:

- Bug reports: open a GitHub issue with a minimal package fixture and the Release Doctor version.
- Feature requests: open a GitHub issue describing the npm release problem the tool should detect.
- Security issues: follow `SECURITY.md` and report privately.

Before opening an issue, run:

```sh
npx release-doctor --json .
```

Include the relevant issue codes and the smallest package layout that reproduces the problem.
